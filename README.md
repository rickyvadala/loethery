# Lo{ether}y
 The web3 lottery

### Install
```shell
npm install
```

### Environment
- CREATE an etherscan account
- CREATE an apikey
- CREATE a .env file
```
// inside .env file
VITE_ETHERSCAN = YOUR_API_KEY
```

- You can access them like this
```javascript
console.log('VITE_ETHERSCAN: ', import.meta.env.VITE_ETHERSCAN)
```

### Run
```shell
npm run dev
```
